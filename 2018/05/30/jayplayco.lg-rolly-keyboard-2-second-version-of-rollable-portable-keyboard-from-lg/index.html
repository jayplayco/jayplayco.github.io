Hunter’s comment

If you are using a laptop, tablet or even a bigger smartphone, sometimes you need to write longer text on this products. A laptop should be not a problem, but as soon as we start to type longer text on touch screens my productivity is going down quite fast.

Bluetooth keyboards are normaly very bulky, so I don`t carry them around but this Rolly Keyboard from LG can be rolled and it means that the product is really small and easy to carry around.

Based on Bluetooth HD profie and a rolled out dimension of 10.78”x1.24”x1.20” it runs up to 3.5 month with one AAA battery.

The first version of this product was quite expensive for a bluetooth keyboard and the official retail price is 100USD, but you can get it now also on amazon for about 45USD, which is OK.
