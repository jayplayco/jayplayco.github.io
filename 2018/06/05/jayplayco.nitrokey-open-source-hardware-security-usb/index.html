Nitrokey - Open Source Hardware Security USB

Nitrokey

Open Source Hardware Security USB

Screenshots

![how-nitrokey-works-min.jpg](https://i.imgur.com/OiwPNKh.jpg)

Hunter’s comment

After looking into different types of Open source projects, I finally arrived at Open Source Hardware.

Product explanation

Nitrokey is a USB Stick which can protect emails,files, hardrives, certificates and also online account. This is managed with cryptography storing the secret keys in the Nitrokey hardware. There is a Pin protection and also a protect against loss.

Unique Selling Proposition

- High Security with RSA keys up to 4096 bit and AES-256support - Open Source Hardware - No Backdoor - Hidden encrypted storage
