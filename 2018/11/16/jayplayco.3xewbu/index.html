[스팀픽] 증인 노드 운영

안녕하세요, @jayplayco 입니다.

오랜만에 스팀픽 관련 소식을 들고 왔습니다.

https://steempeak.com/

다름이 아니라 스팀픽측이 어제부로 공식적으로 증인노드를 운영하고 있다고 선포를 했습니다. (물론 서버는 이미 이전 부터 돌고 있긴 했습니다만, 스팀페스트와 같이 공표하는 것을 목표로 해서 이제 공식 발표를 했네요.)

  • 스팀피크 관련 소개 내용을 보시려면
    https://steempeak.com/kr/@jayplayco/steempeak

  • 공식적인 공지는 하기와 같습니다.
    https://steempeak.com/witness/@steempeak/announcement-the-new-steempeak-witness-new-witness-features

1. 왜 스팀픽은 증인을 잘 할 수 있을까?

제가 개인적으로 생각하는 점들을 정리하겠습니다.

  • 스팀픽은 현재 스팀기반의 스팀잇을 가장 잘 표현한 3rd 파티입니다.
  • 예쁘고, 빠르고, 필요한 거의 모든 기능을 지원합니다.
  • 개발진과 팀이 다양한 아이디어로 빠르게 개선점들을 적용해나가고 있습니다.
  • 스팀픽 운영진은 피크몬도 같이 운영합니다. (피크몬com과 피크몬kr)
  • 피크몬에서 나오는 수익 일부가 스팀픽 운영에 쓰여지는 관계로, 우선 서버 운영에 필요한 비용 걱정이 없다
  • 증인 서버에서 나오는 모든 수익은 증인 서버 운영비와 스팀피크 개발 비용에 쓸 예정임.

2. 스팀픽이 뛰어난 것을 보여주는 예시

이번 증인 출시와 같이 스팀픽에서 보여지는 증인 페이지를 전격 개선(증설)했습니다. 그간 증인 보팅및 찾기 등 정말 불편했던 사항들을 말끔하게 정리해서 스팀픽 내부의 한페이지에서 깔끔하게 정리해두었습니다.

https://steempeak.com/witnesses

직접가서 확인하는것이 가장 빠르긴 하지만, 중요한 기능은 하기와 같이 정리합니다.

  • UI 개선 (모든 중요 data표기)
  • 전체 랭킹 표시 가능 (100위까지가 아니라 전체 순위 다 보임)
  • 필터 기능 (증인 활성/비활성, 투표한 증인만 보여주기)
  • 남은 투표권 확인 (로그인시)
  • 투표시 Steemconnect 열어줌 (active 키 입력)
  • 이름으로 찾기 기능
  • 버젼으로 찾기 기능
  • Params 보기 기능 (블록 사이즈, AC budget등)
  • 증인을 돌리고 있는 어카운트의 경우 Profile에 증인 버튼 추가 (증인 하고 계시는 분들은 그냥 스팀픽 자기 프로파일로 링크를 걸어주시면 쉽게 증인 투표를 할 수 있습니다. 오홋!)

3. 서버사양

일부 궁금해하시는 분들이 계셔서 같이 공지해드립니다. ^^

전용 서버 (Dedicated Server)

* CPU: 4 Core @ 3.60 GHz * RAM: 64GB DDR3 RAM * HDD: 2x 250GB SSD * OS: Ubuntu 16.04

소프트웨어

* steemd의 개별 build를 이용해 가능한 가장 빠른 response time 구현. * @therealwolf의 pricefeed script 적용 * 자체 증인 모니터링 소프트웨어 개발로 이메일 경고 시스템 구축

백업 서버

랭킹이 예비에서 증인으로 올라가게 되면 백업 서버 증설 예정임.

4. 팀

![image.png](https://files.steempeak.com/file/steempeak/jayplayco/0ivUI8Me-image.png)
  • @asgarth는 개발을 혼자서 담당(감당? ㅎㅎ) 하시는 분으로 현재 위치는 이태리에서 작업하고 있습니다. 커뮤니티에서 요청되는 기능이 있을때 발빠르게 적용하는 것을 보여줬었죠.
  • @dmytrokorol 폴란드에 있으면서 디자인을 맡고 있습니다. 풍경사진을 전문적으로 찍으시는 분입니다. ^^
  • @jarvie는 제가 피크몬때문에 가장 자주 대화를 하는 친구입니다. 비즈니스적으로 아이디어가 많고 개선사항이나 사용성에 대한 아이디어를 집합해서 스팀픽이나 피크몬에 적용하는것을 추진합니다. 피크몬KR도 이분의 도움으로 현실화가 될 수 있었습니다.
  • @r00sj3 디스코트에서 모이는 커뮤니티를 관리하면서 제품에 대한 궁금증이나 문제점을 파악/답변해주고 있습니다.

5. 투표하기

스팀픽에 대한 믿음이 생기셨다면 증인투표를 하기 위해서 여러가지 방식이 있습니다.

  • 그냥 링크를 클릭합니다. https://steemconnect.com/sign/account-witness-vote?witness=steempeak&approve=1&redirect_uri=https%3A%2F%2Fsteempeak.com%2F%40steempeak
  • 스팀픽 프로파일 페이지가서 투표한다. https://steempeak.com/@steempeak
  • 스팀픽 증인 페이지에서 steempeak을 찾아서 투표한다. https://steempeak.com/witnesses

이상과 같이 스팀픽 증인 출마에 대한 정리를 드렸습니다. 스팀 기반 좋은 댑과 3rd 파티들이 자리 잡아서 더욱더 활기차고 좋은 스팀 환경을 만들어줄 팀들을 상위 랭킹에 올리는 것이 좋다고 생각합니다. ^^
