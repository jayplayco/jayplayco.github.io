[스몬] KR 2nd 토너먼트 시작 (11월 2일 오전10시 한국 시간) - 총 상금 200 스팀 이상임 ^^;

![](https://cdn.steemitimages.com/DQmRqgngGJYGk2Q58MyJXv46oHRctVxRxEuHAESujoZFn3a/image.png)

안녕하세요, @jayplayco 입니다.

이번 스몬 테스트 시즌도 이제 거의다 끝나가네요. 새로운 시즌을 맞이하기 전에 KR 두번째 자체 리그를 모집하고자 합니다.

방식에 관련해서 고민을 많이 했습니다만, 다음과 같은 룰을 적용하고자 합니다.

  • 참가비용 2SBD
  • 참가비용 전송하시는 분들에게는 베타 부스터팩 1개 무조건 증정
  • 참가신청 기간 : 2018년 11월 1일-11월 2일 오전 8시까지 (한국시간) - 2SBD 입금으로 봅니다.
  • 리그 시간 : 11월 2일(금) 오전 10시- 11월 5일 (월) 오전 11시 (만 3일간)
  • 배틀 조건 : 알파던 베타던 레벨 1 소환사 카드만 사용. (신규 유입자 기회를 높이기 위함입니다.)
  • 참가조건 : 1인당 1계정 참가 (부계정으로 하시는것도 상관 없으나 1계정만 참가부탁드립니다.)
  • 토너먼트 기간중 참가아이디는 꼭 레벨 1 소환사 카드만 사용하기를 당부드립니다. (부계정을 하나 새로 파서 시작하는것도 방법이긴 합니다. 메인계정은 스몬 토머먼트 참가하셔야하니까요ㅎㅎ)

상금

* 1등 : 참가비용으로 받은 전체 SBD (예: 10명 참가하면 20SBD) 베타 부스터팩 10개 (참고 : 혹 참가자가 10명이 안될 경우 최소 20SBD가 되도록 운영자가 Steem으로 채우겠습니다.) + * 2등 : 10 Steem + 부스터팩 10개 * 3등 : 5Steem + 부스터팩 5개 * 4등-7등 : lucky2님의 지원으로 **알파** 커먼 골드 카드 한개씩 드립니다. (https://steemit.com/dclick/@lucky2/-kr-2nd-11-2-10--1541032486209)

네, 1위에게 거의다 몰아줍니다. ^^ 우리는 결과가 중요하니까요. ㅎㅎ

익명의 후원자 분께서 200스팀을 후원해주셔서 상위 상금에 추가로 다음과 같이 상금을 추가로 배분합니다.

  • 1위 100스팀
  • 2위 50스팀
  • 3위 25스팀
  • 4위 10스팀
  • 5,6,7위 5스팀

토너먼트 접수 마감

  • 총 23분이 접수해주셨고요 모든 분들께 부스터팩 나갔습니다. ^^ 참가계정으로 보내졌으며, 아직 정확한 참가계정이 밝혀지지 않은 분들의 경우는 (글로리7님, 굿헬로님) 메인 계정으로 부스터팩을 보냈습니다.

상금 최종 정리

  • 1위 : 100스팀 + 46스달 + 10부스터팩
  • 2위 : 60스팀 + 10부스터팩
  • 3위 : 30스팀 + 5부스터팩
  • 4위 : 10스팀 + 알파 커먼골드 한장 (lucky2님 지원)
  • 5위-7윌 : 5스팀 + 알파 커먼골드 한장 (lucky2님 지원)

익명의 후원자분께는 진심으로 감사드립니다. 모든 KR 스모너분들을 대신해서 넙쭉 인사드립니다. 꾸~~벅

점수계산방식

> 지난 1st 리그와 같은 방식으로 진행하겠습니다. **수정** @knight4sky 님의 제안으로 게임방식에 약간의 수정이 들어갑니다. 문제는 시작시 고레이팅 사용자분들은 사실상 1렙 소환사로 점수가 내려갈 수 밖에 없는 구조라서 (스몬 매칭이다 보니까) 1000점 이상인 분들께는 일일이 신규 계정이나 타계정으로 참여가 가능하신지 문의를 했습니다.
  • 2018년 11월 2일 오전 10시에 전체 참가자에 대한 Data를 집계해서 Rating, Battles, Wins를 기입합니다.
  • 확인은 https://steemmonsters.com/players/details?name=자기아이디 으로 합니다.
  • 기입확인 (및 플레이어 확인)은 다음 구글 시트에서 하실 수 있습니다. https://goo.gl/9vLwe9
  • 2018년 11월 5일(월) 오전 11시 토너먼트 종료하겠습니다.
  • 순위 선점방식 계산은 = [종료시 Rating] x (([종료시 Wins]-[현 Wins])/([종료시 Battle]-[현 Battle]))+(([종료시 Battle]-[현 Battle]) x 0.01))
    (KR 토너먼트 최종Rating 점수에 승률을 곱하고 마지막에 배틀수도 점수에 영향을 주도록 했습니다.) <- 이것이 수정된 내용입니다. 결국 최종 rating, 승률과 배틀수를 남기기로 했습니다.

신청방법

* 2SBD를 @jayplayco 에게 보냅니다. * 댓글에 참가 신청을 남기고, 부캐로 신청할 경우 부캐 ID도 같이 넣어줍니다.

많은 참여 부탁드리며 새로운 사용자 유입도 되었으면 좋겠습니다. ^^
