스팀몬스터 변경 사항

플로리다 달팽이님께서 이미 자세하게 알려주신 바가 있어서 하기와 같이 링크를 공유드립니다.
https://steemit.com/steemmonsters/@smongo/steemmonsters--1542226524668

크게는,

  1. 이미 배틀에 쓰인 카드는 선물이나 구매를 통해 순위 배틀에서는 7일동안 쓸 수 없음.
  2. 챌린지, 토너먼트와 연습경기는 상관 없습니다.
  3. 조건변화 : 팀 매칭후 무작위 조건이 주어져서 그에 맞춰서 팀을 구성해야 함. - 현재 상태 그대로 봇 사용은 어려워짐.
  4. 매칭 수정 : 다이아 리그는 다이아 리그끼리만 매칭됨.

피크몬 변경 사항

공식 내용 : https://steemit.com/steemmonsters/@peakmonsters/mobile-insights-speed-update

  1. 모바일 사용성 개선

    • 마켓과 컬렉션 페이지에 있는 이미지 사이즈 수정을 했서 로딩 시간을 줄이고 UI에도 어느정도 수정을 봐서 사용성을 개선했습니다. 이와 관련해서 모바일 사용성 개선에 의견 있으신 분들은 댓글로 남겨주시면 정리해서 피크몬 개발팀에 전달하겠습니다. (@glory7님은 이미 하셨네요 ^^)

  2. 스피드 개선
