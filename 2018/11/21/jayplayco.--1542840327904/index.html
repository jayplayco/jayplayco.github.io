[피크몬] 피크몬 업데이트

안녕하세요 @jayplayco 입니다.

피크몬이 업데이트 되어 업데이트된 사항 전달 드리고자 합니다.

  • 마켓과 컬렉션에서 그리드뷰가 기본으로 보여짐.(데스크탑)

  • 카드 리스트가 페이지로 보여져서 불필요한 스콜을 방지함.

  • 장바구니에서 현재 갖고 있는 금액 확인 가능.
![image-1542840128739.png](https://s3.ap-northeast-2.amazonaws.com/dclick/image/jayplayco/1542840139005.png)
  • 프로파일에서 Depost 버튼 생성. (추후 마켓에서 활용될때 돈을 입금해놓고 카드를 구매할 수 있는 기능을 위해 꼭 필요함)
![image-1542839717829.png](https://s3.ap-northeast-2.amazonaws.com/dclick/image/jayplayco/1542839728147.png)
  • 카드 합칠때 총 BCX와 카드 몇개를 합치는지 보여줌.
![image-1542839899241.png](https://s3.ap-northeast-2.amazonaws.com/dclick/image/jayplayco/1542839909471.png)
  • 스핌픽 로고에 맞게 로고 업데이트
  • 주문 상세사항에 Verify 팝업 표시
  • 컬렉션 페이지 몇몇 카드의 보기 수정
  • 일부 페이지및 팝업 레이아웃 수정
  • 새로운 퀘스트 관련 액티비티 필터 기능 추가
