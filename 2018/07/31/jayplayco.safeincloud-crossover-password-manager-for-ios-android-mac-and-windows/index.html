Hunter’s comment

I had always issues to remember my passwords and needed at the end a software manager where I can put all my passwords and receive them easily.

I am using SafeInCloud for more than 4 years and it is now my only password manager.

  • Crossover platform synchronization from iOS to Windows to Mac,
  • strong 256bit AES encryption and cloud-based storage of passwords. One important part here is that I can choose the cloud from Google Drive, Dropbox and so on. The databank will be saved on my personal cloud, so I don`t have to worry that SafeInCloud manages their server for a DB.
  • From my iOS devices I love to use the Fingerprint Login and on Desktop, there are Chrome plugins (also for Safari, Firefox, and Opera) to catch directly new passwords for websites.

The Desktop version is free and there is also a free version for mobile, but the pro functions are only available for 2 weeks. It would be good invested bucks for function.
