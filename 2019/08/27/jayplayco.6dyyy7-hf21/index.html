[HF21] 스팀 블록체인 아직은 이슈가 있음

![image.png](https://ipfs.busy.org/ipfs/Qmc2WFHn5Vs4Tbfhqj6azPkfXP22SBf2HpgS9CTUMZviSg)

안녕하세요, @jayplayco입니다.

개인적으로는 두번째 하드포크인데, 스팀잇의 글 올리기도 이미지가 안올라가고, 현재 이 글은 busy를 통해서 작성하고 있습니다.

하드포크라는 것이 언제나 테스트를 한다고 해도 실제 블록체인에 적용할때에는 문제가 발생할 수 있는 것인 만큼 스팀의 트위터 계정을 보면서 업데이트 상황을 지켜보는 것이 좋을 것 같습니다.

1. 현재 확인되고 있는 것들은

  • 이전 아예 블록 생성을 안함 (현재는 약 1시간전 임시로 고쳐짐)
  • 스팀잇에서 글 작성시 사진 업로드 안됨
  • 스팀픽은 아예 404에러나옴

2. 일어났던 현상에 대한 설명

  • 기술적으로는 코드상 다운보팅풀에 대한 베스팅값이 에러가 나서 블록 생성이 안됨.
  • 이를 위해 임시적으로 uint64_t를 uint128_t로 상향조정해서 수정
  • 상세한 설명 링크 : https://busy.org/@netuoso/welcome-back-post-hf21

3. 현재 상황

  • 블록 생성은 되고 있음
  • 임시로 코드 수정한 것이라서 아마 추가 수정 코드가 적용되어서 증인 서버에 소스 전달될 것으로 예상
  • 스팀 기반 댑이나 컨덴서들의 경우는 아직 사용에 어려움이 있을 수 있음
  • 현재는 스팀잇보다는 busy를 쓰시는걸 추천드림.

4. 오전 7:40분 업데이트

  • 대부분의 작동은 되는것으로 확인
  • 각종 nitrous기반 보팅봇들은 URL이 틀리다고 하면서 보팅금액 다시 반환
  • 스팀픽도 정상 작동
