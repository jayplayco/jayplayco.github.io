스팀헌트의 첫 IEO는 140% 이상의 달성율로 성공리에 마감되면서 5백만 토큰 이상이 판매가 되었습니다.
스티미언분들의 경우 많이 기다리셨을 첫 스팀 기반 IEO와 실제적으로 진짜 스팀으로 구매가 가능한 IEO가 이제 곧 열리게 됩니다.

첫 IEO때 어려움이 있었던 분들이 있어서 간단하게 정리를 드리겠습니다.

1. IEO란?

Initial Exchange Offering의 약자로, ICO는 각 프로젝트가 직접 하는 것이라면, IEO는 거래소를 통해서 판매를 하는 형태입니다. 스팀헌트의 경우는 이미 첫 IEO를 IDCM에서 성공적으로 마무리를 한 상황이며 두번째 IEO는 스팀이나 원화로 구매가 가능하기 때문에 매력이 있습니다.

2. 프로빗 스팀헌트 IEO 요약

a. 일시 : 2019년 3월 22일(금) 12:00 - 2019년 3월 26일(화) 11:59 => 총 5일
b. 보너스 : 15%
c. 추가보너스 : 원화입금후 PROB 코인구매해서 사면 추가 5% (총 맥스 20% 가능)
d. 보너스 락업 기간 : 3달
e. 프로빗 구매 가능 수량 : 750만개
f. 프로빗 스팀헌트 요약 사이트 : https://www.probit.kr/ko-kr/ieo/hunt-round1/0
g. 스팀헌트 IEO 관련 공식 톡방 : https://open.kakao.com/o/g1odiHhb

3. 프로빗 IEO 준비하기

a. 준비사항

  • 프로빗 가입 : www.probit.kr첫 가입절차는 이메일과 이메일 인증으로만 가능합니다. (제 레퍼럴 들어있습니다. 하하^^)
  • 추가 인증이 필요함 (KYC) : 1) 핸드폰 본인인증 => 여기까지만 해도 IEO는 참가 가능.
    2) 계좌인증 (은행계좌연동) 
3) 증빙서류 인증 (주민증, 운전면허증, 여권) -> 증사진, 증 들고 있는 사진, 서약서 작성(증 첨부 사진). => 여기까지 하셔야지 원화 입금이 됩니다.
    ![](https://cdn.steemitimages.com/DQmNjy5sUG14G1YNyjB9VrVydcrwbGGytai7wnbHWLxXkEA/image.png)

4. 주의사항

스팀헌트 공식 톡방 (https://open.kakao.com/o/g1odiHhb)에 프로빗측에서 나오신 소피아님이 현재 상주하고 계십니다. 궁금한것이 있으면 빠른 답변으로 응대해주십니다.

  • 현재 원화로 입금후 구입을 할 경우 가장 높은 수익률을 기대할 수 있어서 프로빗의 원화 입금 => 프로브 코인 구매=> 스팀헌트 IEO 참가가 이익일것으로 예상됩니다. 하지만 원화 입금은 KYC를 사진까지 찍어서 보내야 하며 (이것도 비교적 귀찮긴 한데, 할만은 합니다…) 그리고 원화 입금후 강화된 입금 정책에 의해 3일후에 입금 처리가 되어서 정말 빨라야 23일이나 24일경에 구매가 가능할것 같긴 합니다. 이때 한가지 걱정되는것은 총 물량이 750만개밖에 안되어서 완판이 빨리 되면 원화 입금만 되고 물량이 없는 경우가 있다는 부담이 있긴 합니다.

  • 암호화폐 입금의 경우는 입금후 곧바로 쓸 수 있기 때문에, 스팀, BTC, ETH, XRP, EOS까지 가능합니다.

5. 꿀 레퍼럴 이벤트

프로빗은 아직 가입 안하신 분들 많이 계실것이라고 생각됩니다. 이번에 프로빗에서 레퍼럴 이벤트까지 진행이 되어서 레퍼럴로 들어와서 IEO 참여시 구매의 1%를 지급을 해준다고 합니다.

프로빗 가입하시고, 마이페이지에서 추천인 제도의 링크를 복사하셔서 스팀헌트 IEO를 프로빗 링크와 널리 알리시면 구매 헌트 토큰의 1%까지 얻을 수 있는 방법입니다.

아직 가입 안하신 분들은 제 레퍼럴로 가입하셔도 환영입니다. ^^ ㅋㅋ

6. 결론

스팀으로 구매가 가능하기 때문에 기대가 큰 IEO 이며 KYC도 핸드폰 인증까지만 해서 귀찮은 것도 별로 없어서 좋은 것 같습니다. 한가지 걱정은 프로빗에서 제공하는 총 헌트 토큰이 750만개밖에 안되어서 지난 IDCM에서 봤을때 100%가 너무 빨리 되면 늦게 오시는 분들은 또 다음 IEO를 기다려야 한다는 것입니다. 프로빗의 경우는 100% 시 추가로 받는다는 것도 없는듯 하여, First come First get인것으로 판단됩니다.
