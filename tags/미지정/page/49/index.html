[아마존] 한국 무료배송 시작

아침에 일어나고 나니 대란이 일어났네요. 주변 지인들이 갑자기 미국 아마존에서 피씨 케이스니 매트리스같은 것들을 주문하기 시작했답니다. 이유인즉, 이전 싱가폴에서 아마존 프라임을 소개하기 전에 얼마 이상 구매시 무료배송을 했었는데, 이 서비스가 한국으로 확대되었다는 것입니다.

현재는 90불 이상 주문시 무료배송이 가능하며, 일부 제품은 크기때문에 (예, TV 65”인치) 배송이 거부되는 경우도 있습니다.

예를 들어 노트북을 하나 담아보겠습니다.

위 사진을 보시면 한국으로 배송 비용이 68.85불이 들어가네요.

위에 보시면 옵션이 하나 새로 생겼습니다. FREE Amazon Global shipping 이라는 항목을 선택하시면 배송비용이 무료가 됩니다.

미국에서는 200불 밑으로 구매를 할 경우 관세를 부과하지 않기 때문에 더욱더 싸게 구매할 수 있구요, 제품 한개가 90불이여야 하는것은 아니고 장바구니가 90불 이상일 경우 가능합니다.

아, 이제 배대지들은 큰일이네요~

금일부터 시작해서 아마존이 언제까지 유지를 할지는 모르는 것이나, 싱가폴의 경우는 아마존 프라임을 시작하기 전 시장을 테스트하는 용도로 썼던 것으로 보입니다. 프라임 도입 이후에는 서비스가 프라임으로 대체되었습니다. 한국도 아마존 프라임이 도입될 가능성이 있는 것으로 보입니다. ^^

안녕하세요 @jayplayco입니다.

이제 스팀이 조금씩 익숙해지면서 점점 재미있어지는 것 같습니다. 스팀헌트와의 인연으로 이 프로젝트에 모드로 참여를 하고 있고, 스팀헌트 관련된 글들도 많이 소개해드리고 있었습니다.


Source : Sticker Mule

초기부터 새로운것을 발견하면 가슴이 뛰기 시작하면서 좋은 사람들도 알게 되어 정말 즐겁게 글을 쓸 수 있는 것 같습니다.
스팀헌트가 그렇고 이번에는 다른 한국의 SMT 프로젝트들이 꿈틀꿈틀 거리고 있어서 그 탄생 과정을 어떻게 하면 지켜볼 수 있을까 하다가 @ayogom님의 @steemhotdeals를 접하면서 자세히 들여다보게 되었습니다.

결국 작일 (6월 28일) SMT 프로젝트 개발관련 분들이 같이 모여서 텔레그램방을 오픈하고 새로운 태그 kr-smt를 열기로 했습니다.
필자의 경우는 non-dev과 non-projectowner이나 텔레그램방에서 반갑게 맞이해주었답니다. ^^

1. SMT란?

![](https://cdn.steemitimages.com/DQmVQhujhT9EMxuduiaBtGfornqM6yNmgP668Tx835nK94H/image.png) [Source : William Iven](https://unsplash.com/photos/dAmHWsRYP9c) @twinbraid님의 6개월전 글이 아직도 유효하며 재미있고 쉽게 써주셨습니다. https://steemkr.com/kr/@twinbraid/smt-01 https://steemkr.com/kr/@twinbraid/smt-02

여기서 단순하게 설명을 하자면 컨텐트 바탕의 토큰 ICO를 할 수 있는 제도입니다. 즉 제가 예를 들어 SMT기반 JAY토큰을 발행하고, jayplayco관련 컨텐츠를 만드시는 분들은 JAY토큰을 받을 수 있게 가능한 것입니다. 그것 다른분들이 jayplayco에 대한 컨텐츠에 “좋아요(업보팅)”를 누르면 JAY 토큰을 지급한다는 것입니다.

음.. 예시로는 아무도 저 토큰을 필요하지도 원하지도 않겠죠? 그러나 사람들이 원하는 컨텐츠를 만드려고 하거나 플랫폼을 만들고자 한다면 스팀의 기반을 이용해서 토큰 이코노미를 만들 수 있다는 것이죠.

정말 깊게 들어가고 싶으신 분들은 SMT 백서를 번역하신 @kimhyuckjun 님의 글을 파고 들어가시는 것도 추천합니다. ^^
https://steemkr.com/kr/@kimhyuckjun/kr-steem-smt

한국에서 진행하고 있는 잘 알려진 SMT를 지향하는 프로젝트중에는 @tasteem 과 @steemhunt 가 있지요, 이미 스팀잇의 델레게이션도 성공적으로 받고 있고 글로벌 진출을 힘차게 진행하고 있는 멋진 프로젝트들이죠. 하지만 이미 성공가도를 달리고 있는 프로젝트 말고 준비중이거나 구상중인 프로젝트들도 많이 있습니다.

2. KR-SMT 태그는?

![](https://cdn.steemitimages.com/DQmThGrwipwpk1VoSpvq9trDyHTdo8eZ5X6TeUfXrL5FdnW/image.png) [Source:Manuel Cosentino](https://unsplash.com/photos/M3fhZSBFoFQ)

기본적인 취지는 SMT 관련된 글들을 태그하자는 것입니다. 물론 앞에 KR을 붙였기 때문에 한국과 관련된 SMT 프로젝트면 좋겠죠. 그렇다고 룰이 따로 있는 것은 아니니 SMT 관련된 글들 쓰실 때 같이 써주시면 좋을 것 같습니다. 현재 이 태그에 참여를 하는 프로젝트는 SMT기반 서비스를 준비중인 @steemhotdeals @trips.teem @devdevil0625 @koinbot 입니다. 더욱더 많은 SMT 프로젝트들이 태그 사용으로 준비중이거나 진행중인 서비스를 알리시면 좋겠습니다.

댓글로 준비중이신 SMT 프로젝트가 있거나 생각중이시라면 남겨주세요. 텔레그램방에 오셔서 다른 개발자분들이랑 얘기하시는것도 좋은 방법인것 같습니다.

각 프로젝트에 대한 소개글을 팀마다 했는데 하기와 같이 정리드립니다.

Font Awesome - Icons. Easy. Done

Font Awesome

Icons. Easy. Done

Screenshots

![font.JPG](https://i.imgur.com/W0JkuMG.jpg)
![font01.JPG](https://i.imgur.com/V7Dut6y.jpg)

Hunter’s comment

0. Repository

https://github.com/FortAwesome/Font-Awesome
Hi, @jayplayco here.

When surfing around the web I`m quite sure that almost everybody has seen the implemented icons in different web pages. Font Awesome is, like the name indicates, an awesome Open source project making it available for everybody and implement it easily into each projects.

1. What is is for? Why Icons and why Font Awesome?

Icons, Logos, etc are an fast and direct way to communicate without knowing the language in behind. You can compress the meaning of a category or an action easily with an icon. The difficulty is that not everybody is an graphic designer and able to create their own icon. On the other side license and ownership about icons makes it difficult to use icons out there. https://cdn.steemitimages.com/DQmdwBC7qHh16rHsnhd2QgDc8ZPtAzpFR3ie33FKsK11azP/image.png

Font Awesome is solving two issues at ones, at it is not making over 1,109 Free Icons in a a lot of different categories available, it also solves the issue that you have to bind the icons as image into your web projects, as it is going to be used as a Font. That is really great (and also saving a lot of bandwidth~!)

2. How to use it?

Font awesome is now in the 5th version and there are different ways to implement it into your projects.
a. Web Fonts with CSS
Use the Free CDN for web fonts and CSS
This is great for persons who dont want or cant host files, like wordpress sites.
b. SVG with JS
Also with Font Awesome CDN.
c. Desktop Use
Download the OTF file to use the fonts on your desktop. Find the Files here. They are in the /advanced/options/raw-svg folder included.
https://cdn.steemitimages.com/DQmPNjs5brcNchJPbDhZd9y9y1r2UyMtAFRfyTtEyXAyUEG/image.png
d. Advanced Use
Font Awesome has prepackaged files also for React and Vue.js. Angular, Ember, Ruby and Phoenix are also in preparation. Find it here

Font Awesome has also a very good categorized overview about their Icons which can be accessed here
https://cdn.steemitimages.com/DQmaqGavLDqJazLvw8cqqD994YGShDzxa1xktQcWNvv9nH9/image.png

3. What is the difference between Free vs Pro?

With the free version you get access to total 1,109 icons mostly in Solid. (Which is by the way more than enough for any usual private and commercial use). With Font Awesome 5 Pro, you get 1,877 More Icons in different styles (Solid, Regular, Light) and a Private NPM Repository and soon a dedicated Pro CDN. The license starts for 60USD per year. https://cdn.steemitimages.com/DQmZi9LwemN2Vyy9jKuNCkzFwHLK1HvFNxC1RmLQFoYgZyv/image.png

4. But there is no Steem Logo~!

Yes, in the original Font Awesome library there is no Steem logo included. But the great thing about Open Source is that it is Open Source :) @bhj has made about 6 month ago an own git version with a Steemit Logo included. You can find the text and explanation [here](https://steemit.com/steemit/@bhj/updating-steemit-logo-in-font-awesome)~! Thanks to @bhj for this great work!.

https://steemitimages.com/0x0/https://steemitimages.com/DQmQ7sBrjpshdDCdVsLYXmuSWnWRWFAQTqGSYQVGP1HFsRh/image.png

5.Conclusion

If you need some icons to explain or compress your meanings Font Awesome is a great place to start, as they can be used either private or commercial and with over 1,000 Icons you can find almost every normal icon you can think of. The icon numbers are also growing steadily and getting updates in regular base, so it should be bookmarked by every interested content creator.
