[Steem] Change of the Signup Screen suggestion (Wireframe)

Hi, @jayplayco here,

As we have for Steem already quite some services that are offering immediate and easy creation of Steem accounts, it would be required to integrate them into the interface of Steemit.com.

I have created a posting in Korean explaining for Newbies how to use these services, but the fastest way would be to have it integrated into the Signup process of Steem.

1. Signup Screen of Steemit.com

  • Link : https://signup.steemit.com/

This is the current Signup Screen of Steemit.com. I have gathered trusted sources that can be used to create immediately Steem accounts. As a wireframe, this part could be added to the HTML with a link to each source.

We could either keep the Steemit.com signup as it is and add only the four trusted sources.

  • Steemhunt Link : https://steemhunt.com/sign-up
  • Steemcoinpan Link : https://join.steemcoinpan.com/
  • Steemyy Link : https://steemyy.com/reg.php
  • dev.supporters Link : https://steemit.wonsama.dev/

@steemitblog would be great if you could implement these sources, so Steem newbies have it much easier to join the Steem economy. The best part is, all of these services are for free from trusted sources and Steemians.

2. Implementation

The page would need to be designed based on the wireframe, but the effort itself should be limited for a web designer. The Signup button should lead directly to the Signup page of each service, and if we could manage it a mouseover or click on the service could show some details about the providers (Dapp, Community, witness, etc.)

3. Personal thoughts

I think these kind of implementations are going fast and easy, as it does not touch the blockchain code itself but only the Frontend of Steemit.com itself. It would definitely help a lot of KR users to join Steem much easier and speed up the process also.

[Coinnews] 모바일 운전면허증과 블록체인의 관계

안녕하세요, @jayplaco입니다.

이번 24일부터 운전자들이 스마트폰 앱으로 운전면허증을 증명할 수 있는 시대가 옵니다.

우선 3대 통신사 본인인증앱인 PASS를 통해서 경창청 면허정보 시스템이 연됭된 모바일 운전명허증이 24일부터 출시됩니다. 이는 결국 패스 앱을 이용해서 추가 신청한 모바일 운전면허증 역시 효력을 발생하게 된다는 것입니다.

문제는 도로교통법상에는 효력에 대한 규정이 아직은 없었으나, 정부가 임시허가를 해줘서 사용화가 이루어질 수 있었습니다.

운전면허증은 개인정보 유출의 위험이 분명히 있는 관계로 관련 정보는 스마트폰 내부에만 저장/기록을 하기로 했고 아마존의 블록체인 기술을 이용해서 데이터 암호화를 해서 위,변조에 대한 해킹을 방지하겠다고 합니다.


[출처: 통신3사 Pass 확인 서비스]

이전 뉴스를 확인해보면 각 3사는 패스와 확인 서비스를 하기 위해서 프라이빗 블록체인을 구축하고 경찰청과 도로교통공단의 운전 면허정보 검증 시스템과 연동한다고 합니다.

개인적인 생각


출처: AWS 서밋온라인 코리아 - zdnet korea뉴스

우선 모바일 운전면허증은 카드형 면허증과는 달리 모든 정보를 한꺼번에 보여주지는 않습니다. 일정한 기간동안 유효한 QR코드를 생성하고 이에 대한 정보를 블록체인과 중앙 서버에 검증해서 맞는지 아닌지를 확인하는 서비스를 AWS 프라이빗 블록체인과 연동해서 돌리는 서비스인 것입니다.

즉, 실제로 운전중 운전면허증 확인을 위해서는 쓸 수 있으나, 어디가서 신분증 대용으로 쓰기에는 아직 무리가 있는 것입니다. 또한 스샷을 보시면 아시겠지만 이름과 사진정도가 나와있을 정도이지, 생년월일이나 주소같은 정보는 전혀 보여지지 않고, 확인용 QR코드나 바코드도 일시적으로 유효한 것이라 상당히 안전하게 쓰일 수 있는 서비스가 될 것으로 보입니다.

이렇게 하면 블록체인 기술활용하여 최초로 모바일 운전면허증이 존재하는 국가가 되는 것이라는 생각이 잠시 들면서, 그것보다는 일반적인 블록체인 기술 자체가 그동인 지속적으로 언급했던 우리의 일상생활로 스며들면서 일상화 되는데 한발 더 가까이 오는 것이 더 흥미롭다는 느낌입니다.

[Coinnews] 비트토렌트 에어드랍 중단

안녕하세요, @jayplayco입니다.

트론 관련 토큰들의 경우는 TRX를 들고 있는 유저들에게 에어드랍을 많이 해주기로 잘 알려져있습니다. BTT가 그랬고, JST 역시 그렇습니다. BTT의 경우는 지속적으로 에어드랍이 될 것으로 보였으나, 지난 2019년도를 시작으로 총 16회에 걸쳐서 에어드랍을 하고 이번 5월 11일을 마지막으로 에어드랍을 멈추기로 했습니다.

남아있는 74,002,500,000 BTT의 경우는 비트토렌트 파운데이션에서 비트토렌트 커뮤니티를 위해서 우선은 홀딩하고 남겨둘 예정입니다. 즉, TRX 홀더 에어드랍보다는 좀더 직접적으로 BTT 사용자들에게 우선적으로 보상을 줄 수 있는 방향으로 가고자 하는 것으로 결정했습니다.

보상 방향은 다음과 같습니다.

  • 파일 저장 공간
  • 액티비티 트랜스미션
  • 액티비티 시딩

즉, 파일 공유에 관련하여 공헌을 한 사용자에 대한 보상이 될 것으로 보입니다. 상세사항이 나오게 되면 비트토렌트 파일 공유만으로도 BTT를 보상으로 받을 수 있는 세상이 올수도 있겠네요.

[Steemit] 스팀잇 뉴비 무료 계정 생성 방법

안녕하세요, @jayplayco입니다.

스팀을 사용하고자 하면 우선 계정이 필요한데, EOS와 비슷하게 계정 생성하는것이 그냥 되는 것이 아니라 비용을 지불해야지만 생성이 가능한 시스템입니다.

스팀잇.com에서 직접 생성할 경우에는 보통 2일-7일까지의 시간이 소요될 수 있기 때문에 성격이 급한 한국분들의 경우는 특히 답답할 수 있을 겁니다.

그래서 스팀잇에 관심이 있으신 분들이 빠르고 쉽게 무료 계정을 곧바로 생성할 수 있는 방법들을 소개합니다. 하기 방법으로 생성된 계정들은 3rd 파티의 무료 생성 서비스를 이용해서 만들기 때문에 마스터키를 꼭 변경하기를 권장합니다.

  • 마스터키 변경 방법

1. 스팀 탑 증인 @justyy의 계정생성

  • 주소: https://steemyy.com/reg.php
  • 요구사항: 이메일
  • 계정생성: 즉각 (이메일 인증후)
  • 특이사항: 마스터키 이메일로 비암호화 전달

이메일 확인을 통해서 마스터키를 이메일로 공유해줍니다. 생성후 마스터키 변경을 추천합니다.

2. 스팀코인판

  • 주소: https://join.steemcoinpan.com/
  • 요구사항: 구글 이메일
  • 계정생성: 즉각
  • 특이사항: 비밀번호 직접 설정 가능

비밀번호를 직접 설정하는 관계로 DB에 마스터키가 남을 확률이 높지는 않습니다.

3. 스팀헌트

  • 주소 : https://steemhunt.com/sign-up
  • 요구사항 : 핸드폰 인증
  • 계정생성: 즉각
  • 특이사항: 한달간 소량 임대 (약 15SP) 해줌

핸드폰 인증이 있어야 하는 관계로 휴대폰 번호당 계정이 하나씩 만들어질 수 있습니다. 또한 초기 한달간 활동을 가능하게 하기 위해서 소량 스파 임대도 같이 들어가는 특징이 있습니다.

4. @dev.supporters 팀의 카카오톡 연동 계정 생성

  • 주소: https://steemit.wonsama.dev/
  • 요구사항 : 카카오톡 계정
  • 계정생성: 즉각
  • 특이사항: 최소 한달간 10SP 임대기능

@wonsama님이 만들어주신 스팀 한국 탑 증인인 @dev.supporters에서 제공하는 계정 생성 솔류션입니다.

5. 스팀잇의 계정 생성

  • 주소 : https://signup.steemit.com/
  • 요구사항: 핸드폰 / 이메일 인증
  • 계정생성: 2일-7일
  • 특이사항: 가장 안정적이긴 하나 계정 생성 시기가 즉각적이지 않음.

6. 개인적인 생각

입맛에 맞는 방법으로 계정을 생성하면 될 것 같습니다. 마스터 키워드를 직접 설정할 수 있는 스팀코인판의 구글 연동 솔루션이 외부 사용자로 가장 쉽게 접근이 가능할 것 같고, 그외에는 카톡 연동이 되는 @dev.supporters의 경우도 소량 임대까지 되어 있어서 곧바로 글 쓰기를 원하는 분들에게는 적합할 것으로 보입니다.

이것 저것 다 남기는것이 싫을 경우 @justyy분의 솔루션은 이메일주소 아무거나 하나 있으면 계정이 생성이 되나, 초기 임대가 없어서 처음 활동에 제약이 조금 있을 수 있습니다.

그중에서도 뉴비들을 전문적으로 찾아서 큐레이션하는 커뮤니티팀을 운영중에 있습니다.

  • @cmp2020 (USA)

  • @kiwiscanfly (New Zealand)
  • @mariita52 (Venezuela).

전반적인 활동 자체는 큐레이터 팀이 자립적으로 결정하고 진행할 수 있도록 할 예정이라고 합니다.

안녕하세요, @jayplayco입니다.

특히 새롭게 들어오신 분들이나 스팀에 정착을 원하시는 분들은 @steemitblog 계정을 팔로우하시면서 거기서 나오는 현재의 이벤트들을 잘 들여다보시기 바랍니다. 오늘의 일상은 스팀잇에서 실행하는 이벤트에 대한 소개를 해보고자 합니다.

https://steemitimages.com/640x0/https://cdn.steemitimages.com/DQmXrkvRCpHj62gm5XR1TUDxA7R7Kyybi2FgkTaRrPzgcoP/steemit-logo.png

블로그의 내용은 전부다 영어로 되어 있지만, 일반적으로 이벤트를 하는 것은 모든 언어를 아우르기 때문입니다.

예를 들어 현재 ‘나의 일기’ 콘테스트를 일주일째 진행중에 있습니다만, 한국 일상 생활 블로깅 유저분들은 아직 참여를 많이 안하시는 것 같습니다. (필자는 보통 암호화폐 관련 글들이라 주제와는 잘 어울리지는 않아서요)

  • 원문: https://steemit.com/the100daysofsteem/@steemitblog/100-days-of-steem-day-65-the-diary-game-update-3-prize-news

해당 내용은 자신의 일상생활에 대한 포스팅을 올리시고 맞는 태그를 다시는 것으로 참여가 완료가 됩니다. 나중에 점수제를 통한 랭킹 보드를 통해서 상금도 주어질 예정입니다. 물론 그동안 이미 일주일간 진행이 되어서 늦은 감이 있을 수 있으나, 스팀잇 재단에서 운영하는 큐레이션 팀에서 큐레도 같이 들어가는 관계로 특히 뉴비분들이나 일상생활 블로깅하시는 분들에게는 좋을 것 같습니다.

1. 상금 내용

각 포스팅에 대한 점수제로 총 점수를 추후 종합해서 상금을 나눠줌. 현재 랭킹은 스팀잇블로그 계정을 통해서 확인이 가능함.

  • 1st place - 1000 STEEM
  • 2nd place - 750 STEEM
  • 3rd place - 500 STEEM
  • 4th place - 250 STEEM
  • 5th place - 125 STEEM
  • 추가로 5명에게 각각 25스팀

2. 참여 방법

일상생활 포스팅을 하고 다음과 같은 태그 2를 달아줍니다.

  • #the100daysofsteem #thediarygame
  • @steemitblog 100days of Steem 포스팅중 최신 포스팅에 가서 자기글 링크를 달아둡니다.
  • 글 제목에 일기의 날짜를 입력합니다.
  • 글은 steemit.com에서 작성을 하도록 한다.

3. 일정

이미 일주일이 지났다고 하나, 7월 11일까지 진행될 예정이라서 빠르게 추격해서 점수를 쌓으실 수 있는 분들이 몇몇 계실 것으로 짐작됩니다. ^^

4. 점수를 받는 방법

  • 콘텐츠 퀄러티 : 주관적인 기준으로 콘텐츠에 대한 점수를 1-5로 주어집니다.
  • 프로모션 포인트: 다른 SNS에 자기의 포스팅을 얼마나 잘 홍보하는지를 기준으로 합니다. 즉 페북이나 트위터, 카카오톡, 블로그 등에 홍보를 했을 경우, 이에 대한 스샷을 @steemcurator01이 와서 큐레를 하기 전에 댓글로 스샷을 남겨야 점수가 적용됩니다.
  • 댓글 포인트: 다른 사람의 포스팅에 좋은 댓글을 많이 남길 경우 추가 점수를 받을 수 있습니다.
  • 다른 스티미언이 다이어리 게임을 자신때문에 시작하게 되면 최대 2점까지 추가로 받을 수 있습니다.
  • 프로모션 플러스 포인트: 다른 사람들의 다이어리 게임 포스팅을 홍보해주면 추가 점수를 받을 수 있습니다.

큰 그림으로는 가장 간단하게 태그를 남기고 콘텐츠 퀄러티 기반과 댓글 활동 기반으로 점수를 받을 수 있다고 판단하시면 될 것 같습니다. 또한 재단에서 임대를 통해 큐레이션 활동을 하고 있는 @steemcurator01 계정의 큐레도 콘텐츠에 따라서 오기 때문에 활동이 할만 할 겁니다.

많은 일상 블로깅 스티미언들과 맘카페 스티미언분들등의 참여가 독려됩니다. 참여하시면 제 글 보고 하셨다고 한마디씩은 땡큐 입니다. ㅎㅎㅎ
