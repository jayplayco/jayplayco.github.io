Now with the new JJM bot, we are able to vote automatically based on the following rules.

0. TL;DR

  • You need more than 100 JJM to get a daily upvote.
  • For an upvote you need to add the JJM tag

1. Voting range

  • Max vote will be from now 1+45%=46%
  • For each 1000JJM hold you will get additionally 1%
  • This means for 45K JJM tokens you will get a 46% upvote, but also for 100K JJM tokens you will get a 46% upvote.

  • Our JJM bot will vote once a day.
  • If you use more than one JJM tag within that 24hours window, only one post will be upvoted.

3. Min. needed JJM

  • The minimum needed JJMs are 100 tokens to be able to get an upvote from the JJM bot.
  • Please remind that only the Qty of JJM that are currently not under an open order is counted.

4. Additional function

  • We are preparing to leave a comment after voting.

5. Disclaimer

  • Please be informed that these rules can change at any time, especially when the main account @virus707 is adding additional SP or the holder numbers are increasing.

![image.png](https://files.steempeak.com/file/steempeak/jayplayco/cC7aSglZ-image.png) Gold vector created by freepik - www.freepik.com

안녕하세요, @jayplayco 입니다.

어떤식으로 글을 쓸까 고민하다가, 그냥 지금 있는 그대로, 제가 생각하는 사항들을 적어볼까 합니다.
발단은, 스팀엔진 (https://steem-engine.com/?p=market&t=JJM) 이라는 곳이 생겨나면서 저같은 코알못(코드를 모르는 사람)도 10분만에 5만원을 들여서 코인을 생성할 수 있는 스팀생태계가 되어서 일어나는 현상에 대한 것입니다.

주의 : 이 글은 절대 투자자문이 아닙니다. 단지 현재 스팀상에서 스팀엔진의 쉬운 토큰 생성으로 인한 토큰 생태계가 탄생하는 과정을 지켜보는 글입니다.

  • 발행설명 : JJM 태그가 없어서.
  • 토큰 설명 : (현재까지는) 기능은 없고 토큰 이코노미만 있는 토큰, 코인홀더와 고민후 방향 개척.

2. 개인적인 생각

2019년 3월 19일 기준

  • @virus707님의 스파 : 440K STEEM (그중 현재 임대 201K이나 금년중 회수 예정)
  • 440K SP 업보팅시 11.19STU
  • Virus707 JJM토큰 구매시 -> 11STU x 10(회) x 7(일)x 25%(큐레이션)=192.5STU
  • JJM 토큰 보유자 구매시-> 11STU x 10(회) x 7(일)x 75%(Author)x50%(SBD분)=288.75STU
  • 일주일에 토큰 이코노미 내에서만 총 481.25STU의 시장가 매수가 가능함.
  • jjangjjangman 태그를 JJM 토큰 보유에 따른 JJM 태그로 전환 가능성 있음.
